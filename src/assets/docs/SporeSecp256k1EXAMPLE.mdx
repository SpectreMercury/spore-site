<MarginContainer bottom={48} />

<NoteBoard header={'üç° Ingredients:'} contents={['Node.js version 18.0.0 or above', 'PNPM version 8.0.0 or above', 'Terminal', 'Spore-SDK repository']}/>

<MarginContainer bottom={48} />

#### üçÑ Step 1: Environment Setup

<MarginContainer bottom={40} />

1\. Open your terminal and clone the spore-sdk repository:

<MarginContainer bottom={16} />

```bash
git clone https://github.com/ShookLyngs/spore-sdk.git
```

<MarginContainer bottom={32} />

2\. Navigate to your spore-sdk directory (make sure to replace the pathname with yours)

<MarginContainer bottom={16} />

```bash
cd ~/projects/spore-sdk
```

<MarginContainer bottom={32} />

3\. Install dependencies by running the command:

<MarginContainer bottom={16} />

```bash
pnpm i
```

<MarginContainer bottom={32} />

4\. Once installed, build your local packages with:

<MarginContainer bottom={16} />

```bash
pnpm run build:packages
```

<MarginContainer bottom={16} />

Your environment setup is now ready if all commands execute without errors!

<MarginContainer bottom={48} />

#### üçÑ Step 2: Explore the Example

<MarginContainer bottom={32} />

1\. In your terminal, navigate to the Spore Secp256k1 examples directory:

<MarginContainer bottom={16} />

```bash
cd examples/secp256k1
```

<MarginContainer bottom={32} />

2\. You will find the example code stored in examples/secp256k1/apis. Feel free to experiment with the code.
<MarginContainer bottom={32} />

#### üçÑ Step 3: Construct and Send a Spore Transaction

<MarginContainer bottom={32} />

1\. To create a spore on-chain, execute the apis/createSpore.ts example by running the command:

<MarginContainer bottom={16} />

```bash
ts-node apis/createSpore.ts
```

<MarginContainer bottom={32} />

2\. This script will construct and send a transaction to create a spore on-chain. If successful, a Transaction Hash will be returned. You can use this hash to review your transaction details on the CKB Explorer.
